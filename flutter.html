MRT Compass Flutter Project

A Flutter-based MRT companion app for home screen, travel estimation, and real-time transaction management.

My contributions to the development of the MRT Compass Flutter project include creating the Home Screen, assisting my teammates in building the transaction backend, developing the Travel Estimation feature, and creating the FAQ section.

Home Screen Development

I was responsible for coding the Home Screen Page using Flutter. My focus was to design an intuitive and interactive user interface (UI), ensuring smooth navigation and implementing essential components such as buttons, icons, and layouts that support the app’s main functionality. The entire page was built to provide a responsive and engaging user experience across different devices.

Backend Development and Travel Estimation Feature

For the transaction backend, I collaborated with my teammate to develop the backend system using Firebase. We configured Firebase to handle user authentication, store transaction data, and monitor transaction history in real-time. This collaboration ensured that every transaction is securely recorded and can be easily accessed anytime by users.

I also contributed to the Travel Time Estimation feature using Flutter. I designed the calculation logic based on real-time data such as train schedules, selected routes, and MRT operational conditions. The user interface was designed to be simple and intuitive, allowing users to easily select departure and destination stations. The system continuously updates and displays accurate, real-time travel time estimates while maintaining smooth performance.

Virtual FAQ

In this section, I developed a virtual FAQ system designed to respond to user needs quickly and efficiently. Visual elements such as friendly chat bubbles and clear action buttons including Help, Contact Us, Check Balance, and Transaction History, ensure that users can instantly access the information they need. The interface was built with a strong focus on usability, allowing users to either type their questions directly into the provided input field or choose from the predefined options. The well-organized layout and vividly colored buttons provide intuitive access to the main features, enhancing both user experience and interaction flow.
